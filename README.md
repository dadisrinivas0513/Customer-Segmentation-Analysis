# Customer Segmentation Analysis Using K-Means Clustering

## Project Overview

This project analyzes mall customer data and segments customers into distinct groups based on Annual Income and Spending Score using the K-Means Clustering algorithm.

## Objectives

- Analyze customer behavior
- Perform exploratory data analysis (EDA)
- Apply K-Means Clustering
- Generate customer segments
- Provide business insights for targeted marketing

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

## Dataset

Mall Customers Dataset containing:
- Customer ID
- Gender
- Age
- Annual Income
- Spending Score

## Machine Learning Technique

K-Means Clustering

## Results

Five customer segments were identified:
- High Income, High Spending
- High Income, Low Spending
- Low Income, High Spending
- Low Income, Low Spending
- Average Customers
